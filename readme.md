## Sobre

**Fath Spot** é uma ferramenta avançada desenvolvida em PowerShell que modifica o cliente desktop do Spotify para Windows. Ele não apenas remove anúncios e bloqueia podcasts indesejados, mas transforma o cliente em uma central de música completa com capacidade de download.

## Funcionalidades

### experiência Limpa
- **Block Ads:** Bloqueia anúncios de áudio e visuais (patch no `xpui.js` e binário).
- **No Podcasts:** Remove seções de podcasts da tela inicial para focar na música.
- **Restauração:** Sistema de backup automático que permite restaurar o Spotify original a qualquer momento.

### Ferramentas de Download
- **Baixar MP3:** Integração automática com **yt-dlp** e **FFmpeg** para baixar músicas com metadados e capa em alta qualidade.
- **Lyrics Grabber:** Busca e salva letras de músicas utilizando múltiplas fontes (Lyrics.ovh, Musixmatch, YouTube Music).

## Instalação

1. Certifique-se de ter o **Spotify Desktop** instalado (versão exe/site oficial).
2. Baixe este repositório.
3. Execute o arquivo `FathSpot.bat` (ele solicitará permissão de administrador automaticamente).

## Como Usar

O script possui um menu interativo fácil de usar:

1. **Remover Ads:** Aplica os patches essenciais.
2. **Remover Podcasts:** Limpa a interface.
3. **Mod Completo:** Aplica ambas as remoções.
4. **Ferramentas:** Acesse o menu de downloads de música e letras.
